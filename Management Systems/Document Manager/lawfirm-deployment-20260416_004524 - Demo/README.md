## Quick Start (from this folder)

1. **Configure server environment**
   - In `server/`, copy `DEPLOYMENT_ENV_TEMPLATE.txt` to `.env`
   - Replace ALL placeholders (JWT_SECRET, VAULT_MASTER_PASSWORD, MFA_ENCRYPTION_KEY, DB_ENCRYPTION_KEY) with generated values
   - Generate values: `node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"` (run 4 times for each secret)

2. **Install production dependencies**
   ```powershell
   cd server
   npm install --omit=dev
   cd ..
   ```

3. **Start Master Vault first** (required before zenlaw-server)
   ```powershell
   cd master-vault
   npm start
   # Or: node src/index.js
   # Verify: curl http://localhost:3333/health → {"status":"ok"}
   ```

4. **Start all services with PM2**
   ```powershell
   pm2 start ecosystem.config.js
   ```

5. **Verify**
   - ZenLaw server: http://localhost:5002
   - Master Vault: http://localhost:3333
   - Rotation scheduler: running (Python)
