Workstation Management

Monitor and manage all connected workstations from a central location with secure sync

Centralized Workstation Control

Each desk runs the Workstation Monitor agent: it mounts a virtual drive, keeps client folders in sync with the server, and answers “open this document” requests from the web app. The server is the firm’s encrypted hub—files may arrive from any workstation upload or monitored folder—and every other machine downloads decrypted copies to its virtual drive for fast local access.

  • Real-time workstation monitoring and connection status
  • Virtual drive mount (e.g. Z:) mapped to a local sync folder such as C:\LAWFIRM\clients\
  • Bidirectional sync: upload changes from the virtual drive; download updates from encrypted server storage
  • Polls for open requests from the browser and launches files with the default or assigned program
  • Decrypt-on-download so staff edit normal files while the server keeps AES-256-GCM at rest
  • Monitored-folder intake: new client folders and loose files can be discovered and uploaded automatically
  • File locking, conflict handling, and activity logging across the network

Workstation Monitor & open-on-desktop

When someone selects a document in the web app, the request goes to their workstation—not as a browser download, but as an open command on the virtual drive. The monitor checks whether the file is already local; if not, the scheduled sync pulls it from the server’s encrypted store (including copies that originated on another desk) and saves a decrypted working file ready to open.

  • Open-request queue between browser and workstation (typically within one sync/poll cycle)
  • Opens from {drive}\clients\{client}\{filename} with optional per-type program mapping
  • Background sync from server: list all documents, download missing or updated files, decrypt to virtual drive
  • Background sync to server: encrypt and upload edits from the virtual drive
  • File activity and transfer logging in the monitor and Security Center
  • System tray GUI, heartbeats, and alerts when clients or uploads need attention
