Powerful Modules

Extendable modular architecture with specialized modules for every business need

User Management Module

Comprehensive user management with role-based access control, PIN authentication, and automatic lock screen functionality for enhanced security.

  • Create, update, and delete users with full profile management
  • Three role levels: Admin, Manager, and Cashier with distinct permissions
  • Individual PIN authentication for each user (4-8 digits, configurable)
  • Automatic lock screen after configurable inactivity timeout (60-3600 seconds)
  • Role-based settings locking to prevent unauthorized access
  • PIN session management with configurable timeout
  • Default PIN support for users who haven't set up their own PIN

Role Permissions:

  • Admin: Full system access, user management, all settings
  • Manager: Limited settings access, inventory, reports, no user management
  • Cashier: Basic POS operations only, no settings or reports

Key Features:

  • Secure PIN hashing using bcrypt
  • Session persistence across page refreshes
  • Manual lock button in top navigation
  • Settings access control based on user roles

Advanced Reporting Module

Enhanced reporting and analytics with advanced filtering, product performance analysis, customer behavior tracking, and comprehensive export capabilities.

  • Product performance analytics with sales trends and patterns
  • Customer behavior analytics and purchase history tracking
  • Inventory movement reports with detailed tracking
  • Period comparison reports (day, week, month, year)
  • Custom date range filtering for flexible reporting
  • Export to CSV/Excel for external analysis (up to 10,000 rows)
  • Configurable report settings and preferences

Report Types:

  • Sales reports with revenue trends and payment method distribution
  • Product performance reports identifying best sellers
  • Inventory movement reports with detailed tracking
  • Customer analytics with purchase history and patterns

Configurable Settings:

  • Enable/disable product analytics
  • Enable/disable customer analytics
  • Enable/disable inventory reports
  • Enable/disable comparison reports
  • Default date range selection (7, 30, 90, 365 days, or custom)
  • Optional scheduled report generation

Clock In/Out Module

Comprehensive time tracking system for employees with clock in/out functionality, timesheet management, and automatic hour calculations.

  • Automatic clock in/out prompts on login/logout (configurable)
  • Manual clock in/out from timesheet page
  • Optional automatic clock in/out without prompts
  • Active session tracking prevents duplicate entries
  • Optional notes support when clocking in or out
  • Automatic calculation of total hours worked
  • Date range filtering for timesheet viewing

Admin/Manager Features:

  • View all users' timesheets with user filtering
  • Edit clock in/out times and notes
  • Delete timesheet entries
  • Bulk operations for managing multiple entries

Configurable Settings:

  • Show/hide clock in prompt on login
  • Show/hide clock out prompt on logout
  • Auto clock in/out on login/logout
  • Require clock out before new clock in
  • Allow manual clock in/out operations
  • Enable timesheet editing for admins/managers
